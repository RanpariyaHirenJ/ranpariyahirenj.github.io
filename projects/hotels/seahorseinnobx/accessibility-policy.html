Accessibility Policy

Seahorse Inn & Cottages aims to provide a standard website navigation interface for all users including with disabilities. To this end, we continually strive to adhere and comply with World Wide Web's Consortium's Web Content Accessibility Guidelines 2.0 Level AA (WCAG 2.0 AA) on our site. Keeping in mind best practices for accessibility, we have implemented the following features/technology.

  • Clear clean website coding using HTML 5 and CSS
  • Easy page navigation with a clear structured layout
  • Clearly readable Lato font with for ease of reading
  • Large H1, H2, H3, H4 tags for headers
  • Descriptive ALT tags in image content for the visually impaired
  • Clearly visible booking buttons, phone numbers with tel: attributes

Please be aware that WCAG 2.0 AA Guidelines are constantly updated and our compliance is an ongoing process. If you notice any issues with our website accessibility features please call us directly at 252-441-5242 to voice your concerns. We will make every effort to update any feature you may need.
