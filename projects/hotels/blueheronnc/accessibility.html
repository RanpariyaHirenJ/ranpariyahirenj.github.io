Accessibility

This website, blueheronnc.com (also referred to as site, website) aims to provide a clean website interface for all users including those with disabilities. Our website is working towards complying with the World Wide Web's Consortium's Web Content Accessibility Guidelines 2.0 Level AA (WCAG 2.0 AA). While complete compliance is an ongoing process, we have implemented and adhered to most ADA guidelines to make content on this website easily accessible.

  • Colors - We have implemented high contrast text/background ratio colors for easy reading
  • Captions for audio/video wherever applicable
  • Clearly defined ALT tag descriptions
  • Large font sizes for H1, H2, H3 tags for easy reading
  • Large text font font size for content/text
  • Links, both internal and external have clear text descriptions
  • Easy and clean navigation without the need for a mouse
  • Contact telephone numbers with clear tel attributes for easy dialling
  • Labels on data fields clearly defined
  • Fast pagespeed optimizations to minimize site load time
  • White page backgrounds with black text for easy reading

Please note that WCAG 2.0 adherence guidelines are constantly evolving and our compliance process is a continual process. If you find or experience any problems with our accessibility features please do not hesitate to contact us by emailing us at info@blueheronnc.com
