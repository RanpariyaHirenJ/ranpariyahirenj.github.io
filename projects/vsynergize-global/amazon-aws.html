Amazon AWS

Amazon Web Services (AWS) is a comprehensive and leading cloud services platform which can build and run any application in the cloud.

It is a secure cloud services platform which offers a combination of infrastructure as a service (IaaS), platform as a service (PaaS) and packaged software as a service (SaaS) offerings. It offers services to build sophisticated applications with more scalability, flexibility and reliability.

The AWS Cloud covers 60 Availability Zones in 20 geographic regions around the world. These Availability Zones offer AWS users a much more effective and easier way to develop and use applications and databases, making them more available and scalable than usual single datacenter infrastructures or multi-datacenter infrastructures.
